export\+\_\+student\+\_\+grades\+\_\+to\+\_\+directory

Purpose\+: Read a text file mentioned above and write each student\textquotesingle{}s all information in the text file (student name, number of exam scores, the exam scores), the average exam score, and the letter grade to a CSV file in a provided directory. The CSV file name will be \char`\"{}\+Student\+\_\+\+Grades.\+csv\char`\"{}. The delimiter used in the CSV is the comma. If a student\textquotesingle{}s number of exam scores is less than the maximum number of exam scores, then the student\textquotesingle{}s missing exam scores will be filled with an empty column.

For example, if the text file has 2 lines of data\+: SPWKLZWHCIE GAPCEXAS 11 100 88 95 94 98 100 90 71 100 92 98 UIPXMS SLOVX 5 85 90 100 70 98 Then, the csv file will be like below\+: \mbox{\hyperlink{struct_student}{Student}} Name,Number of Exam Scores,,,,,,,,,,,,Average Exam Score,Grade SPWKLZWHCIE GAPCEXAS,11,100,88,95,94,98,100,90,71,100,92,98,93.\+27,A-\/ UIPXMS SLOVX,5,85,90,100,70,98,,,,,,,88.\+6,B+
